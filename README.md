## Install
#### Laravel
 - Install Laravel 
 - Set Up Laravel

#### Get the Code
 - Clone Repository

#### Install all the Stuff!
 - Install Globaly:
 
   `npm install -g browser-sync`
   
 - Run Installers in App directory
 
 ``` 
composer update
npm install 
```

#### Set up Database
- Generate new Database with name like 'a-nicer-way'
- Update your .env with new DB-name and password



## Run the Application
   `npm install -g browser-sync`
